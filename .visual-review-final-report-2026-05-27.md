### Defect Details

**asr-a63l** — Sticky Header Overlap  
When scrolling skill detail page, content scrolls behind the 69px sticky header (content top = -227px).

**asr-4e0** — Empty Search State Missing Action  
Search for non-existent skill shows message "No skills found. Try a different search term or clear the active filter." but NO clickable action button to clear search. User must manually clear the input.

**asr-upoq** — Publish Form Missing Field Validation  
Publish form fields (Registry owner, archive, SKILL.md) have no asterisks (*), no "required" labels, and no inline validation error messages when fields are empty/invalid.

**asr-0743** — Review Validation (INCONCLUSIVE)  
Initial page load of /review/sub-1042 does NOT show premature rejection error. Previous sessions reported "A rejection reason is required before rejecting" appearing before user interaction, but this is not visible now. Full rejection workflow cannot be tested with current mock data.
